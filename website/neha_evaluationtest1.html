Tags in HTML begin and end in "<" and ">" signs. An HTML element is the entire start and end tags, content, and attributes.

Block elements begin a new line on the page and fills the entire width of the page until there is no space left. Some examples are p tags, ul tags, or section tags.

Inline elements display in a line and don't break the flow of the content. An example of an inline element is a link or a span.
